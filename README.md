## Getting Started

### Dependencies

* Latest Bouncy Castle release (>=1.70).
* JRE > 1.6.

### Executing program

* Build the project in an IDE and run org.caulfield.pkiz.PKIZ

* [Soon] Fat Jar to run the program from the Jar
```
java -jar PKIZ.jar
```
